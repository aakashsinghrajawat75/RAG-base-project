# 📄 RAG-Based Document Processing System

A Retrieval-Augmented Generation (RAG) project that processes documents and automatically extracts meaningful information using Natural Language Processing (NLP) and Large Language Models (LLMs).

---

## ✨ Features

- 📂 Upload PDF, DOCX, and TXT files
- 📝 Automatic Title Extraction
- 📖 AI-Generated Document Summary
- ❓ FAQ Generation
- 💬 Question & Answer Generation
- 🔍 Text Preprocessing
- 📚 Multi-Document Support
- ⚡ Fast Information Retrieval

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- LangChain
- Google Gemini API
- FAISS
- NLTK
- spaCy
- PyMuPDF
- python-docx
- pandas

## 📋 Workflow

1. Upload one or more documents.
2. Extract text from the files.
3. Clean and preprocess the extracted text.
4. Generate the document title.
5. Generate a concise summary.
6. Create FAQs automatically.
7. Generate question-answer pairs.
8. Retrieve relevant information using the RAG pipeline.
